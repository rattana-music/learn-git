My First Web Page

นี่คือเว็บไซต์แรกของฉัน และฉันอยากจะทำออกมาให้ดีที่สุด

ฉันจะเรียนรู้การสร้างเว็บไซต์ด้วย HTML, CSS และ JavaScript เพื่อให้สามารถสร้างเว็บไซต์ที่สวยงามและมีฟังก์ชันการทำงานที่ดีได้

สิ่งที่ต้องรู้:

  • HTML เพื่อสร้างโครงสร้าง
  • CSS เพื่อตกแต่งให้สวยงาม
  • JavaScript เพื่อตกแต่งเว็บทำงาน
  • คำสั่ง li ไว้สร้างจุดไข่ปลา
  • คำสั่ง ul การรวมกลุ่มไอเทมแบบไม่มีลำดับ
  • hr ไว้สร้างเส้นคั่น
  • a คือตัวสร้าง Link ต้องมี href="" ไว้ใส่ URL และ target="" ถ้าคลิก Link แล้วเปิดแท็บใหม่

ถ้าฉันมีข้อสงสัย ฉันจะไปหาคำตอบที่: Google
